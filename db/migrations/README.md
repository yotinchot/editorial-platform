Auto-generated by Drizzle Kit. Do not edit manually.

Workflow:
1. Edit schema files in `db/schema/`.
2. Run `pnpm db:generate` to produce a new migration SQL file here.
3. Run `pnpm db:migrate` to apply it against Vercel Postgres.

The migration files are committed to version control so deployments
can apply them deterministically via `pnpm db:migrate` in CI.
